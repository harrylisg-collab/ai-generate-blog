# Blog 项目验收标准

## 1. 功能验收

### 1.1 公开页面
- [ ] Homepage 正常加载，显示博客标题
- [ ] Homepage 显示文章列表
- [ ] 文章详情页正常显示 Markdown 内容
- [ ] 主题切换功能正常 (light/dark)
- [ ] 404 页面正常显示
- [ ] RSS Feed 可访问且格式正确
- [ ] Sitemap 可访问且格式正确
- [ ] 搜索功能正常 (有结果/无结果场景)

### 1.2 认证与授权
- [ ] 登录页面正常加载
- [ ] 正确凭据可以登录
- [ ] 错误凭据显示错误提示
- [ ] 未登录用户访问 /admin 重定向到登录页
- [ ] 未登录用户访问 /admin/posts 重定向到登录页
- [ ] 未登录用户访问 /admin/new 重定向到登录页

### 1.3 管理后台
- [ ] Dashboard 正常加载
- [ ] 可以查看所有文章列表
- [ ] 可以创建新文章
- [ ] 可以编辑现有文章
- [ ] 可以删除文章
- [ ] 可以发布/取消发布文章
- [ ] 可以查看订阅者列表
- [ ] 可以查看用户列表 (admin)
- [ ] 退出登录功能正常

### 1.4 Newsletter
- [ ] 订阅表单在首页可见
- [ ] 可以提交有效邮箱
- [ ] 无效邮箱显示错误提示
- [ ] 重复订阅有适当提示

## 2. 性能验收

- [ ] 首页加载时间 < 3秒

## 风险评估矩阵

| 风险项 | 影响 | 概率 | 严重度 | 缓解措施 |
|--------|------|------|--------|----------|
| 未登录访问 admin | 高 | 中 | 高 | 强制重定向 |
| 数据库连接失败 | 高 | 低 | 高 | 超时处理 + 错误提示 |
| XSS 攻击 | 高 | 低 | 严重 | 输入转义 |
| RSS/Sitemap 失效 | 中 | 低 | 中 | 监控告警 |
| 搜索无结果无提示 | 低 | 中 | 低 | UI 改进 |
